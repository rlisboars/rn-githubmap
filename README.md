React Native Project created during [Rocketseat Bootcamp](https://rocketseat.com.br/bootcamp).

Using react-redux and react-saga just to learn more about theses solutions.

**User Stories:**

- User can see a map
- User can long press in the map and add a github account pin to the map location

**Pending:**

- Allow user to remove accounts
- Allow user to navigate through an accounts list that centers the map in the account selected
- Display issues inside the app instead of an internet browser
